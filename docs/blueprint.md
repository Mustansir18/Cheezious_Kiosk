# **App Name**: Cheezious Connect

## Core Features:

- QR Code Scanning: Scan a Cheezious branch-specific QR code to initiate the ordering process.
- Geolocation Integration: Automatically detect the branch location based on the scanned QR code.
- Mode Selection: Prompt the customer to choose between 'Dine-In' or 'Take Away' options.
- Menu Display: Redirect to the Cheezious menu with item selection and quantity options.
- Order Confirmation: Show order details to confirm items and quantities, with 'Order' or 'Cancel' options.
- Cashier Order Management: Transfer the order to the cashier screen for acceptance and preparation.
- Order Ready Notification: Notify the customer's phone when the order is prepared, prompting them to collect their order.

## Style Guidelines:

- Primary color: Cheerful orange (#FFA500) to reflect the brand's identity and energy.
- Background color: Light, creamy beige (#F5F5DC) to provide a warm and inviting feel.
- Accent color: Complementary dark blue (#00008B) to create contrast for calls to action.
- Font pairing: 'Poppins' (sans-serif) for headlines, 'PT Sans' (sans-serif) for body text, providing a modern and readable combination.
- Use custom icons related to food items and order status for intuitive navigation.
- Clean and straightforward layout with clear visual hierarchy for ease of use on mobile devices.
- Subtle animations for button interactions and order status updates to provide visual feedback.
